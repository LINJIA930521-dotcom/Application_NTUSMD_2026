\# 男七學院 114學年度碩士班甄試錄取名單系統



這是一個為 \*\*國立男七學院 (Nanqi College)\*\* 模擬的碩士班推甄查榜系統。



\## 🎯 專案功能

\- 模擬真實的查榜介面。

\- 自動生成四個系所的錄取名單（正取與備取）。

\- 包含亂數生成的「姓名」與「准考證號」。



\## 🏫 系所與名額概況

1\. \*\*半導體系統工程學研究所 (N701)\*\*

&nbsp;  - 正取：20 名 / 備取：52 名

2\. \*\*高頻脈衝學研究所 (N702)\*\*

&nbsp;  - 正取：12 名 / 備取：31 名

3\. \*\*自旋電子學研究所 (N703)\*\*

&nbsp;  - 正取：6 名 / 備取：13 名

4\. \*\*羅馬拼音碩士學位學程 (N704)\*\*

&nbsp;  - 正取：5 名 / 備取：16 名



\## 🚀 如何使用

1\. Clone 此專案。

2\. 開啟 `index.html` 即可瀏覽。

3\. 或使用 GitHub Pages 進行線上預覽。
